## 🧩 1. Word Search

### 📖 Mô tả

Tìm các từ trong lưới ký tự ( n \times n ) theo 8 hướng:

* Ngang, dọc, chéo

### ⚙️ Thuật toán sử dụng

* DFS (Depth-First Search)
* Greedy Best-First Search (GBFS)
* A* Search

## 🧮 2. Killer Sudoku

### 📖 Mô tả

Giải bài toán Sudoku mở rộng với ràng buộc tổng (cage).

### ⚙️ Thuật toán sử dụng

* DFS (Backtracking)
* A* + Heuristic
